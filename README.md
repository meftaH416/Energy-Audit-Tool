## Troubleshooting

**Photos not saving?**  
- Browser storage might be full - try removing some photos or clear draft

**PDF generation failed?**  
- Check internet connection (needed for PDF library on first load)
- Try on a different browser

**Lost my data?**  
- Data is stored in browser cache - don't clear browser data
- Always download PDF before clearing draft

## Requirements

- Modern web browser (Chrome, Safari, Firefox, Edge)
- JavaScript enabled
- ~10-50MB storage for photos (varies by number of images)

## Privacy

All data stays **100% local** on your device. Nothing is uploaded to any server.
